About This Demo

This application is a demo of a suggested flow for using UPS APIs. In this applicaiton we do a simple shipment. The APIs used are the OAuth, Address Validation, Rate, and Shipment APIs. They are also called in this order. The main points to note are the API calls are made on the Java side, and the frontend simply creates the JSON request, and displays the response JSON.
